# Historial de cambios (Changelog)

Todos los cambios notables en la **Biblioteca Técnica - SVICPDI** se documentarán en este archivo siguiendo el estándar [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/).

El formato de las versiones se basa en [Semanthic Versioning](https://semver.org/lang/es/):

* **MAYOR**: Cambios estructurales o legales críticos.
* **MENOR**: Nuevos manuales o funcionalidades.
* **PARCHE**: Corrección de erratas o enlaces rotos.
