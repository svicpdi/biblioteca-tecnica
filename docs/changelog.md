# Historial de cambios (Changelog)

Todos los cambios notables en la **Biblioteca Técnica - SVICPDI** se documentarán en este archivo siguiendo el estándar [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/).

El formato de las versiones se basa en [Semanthic Versioning](https://semver.org/lang/es/):

- **MAYOR**: Cambios estructurales o legales críticos.
- **MENOR**: Nuevos manuales o funcionalidades.
- **PARCHE**: Corrección de erratas o enlaces rotos.

---

## [1.0.0] - 2026-03-25

### Añadido

- **Sección Baremación:** Nuevo manual para la gestión de candidaturas v2.
- **Componentes:** Integración de diagramas Mermaid para flujos de trabajo.
- **Navegación:** Pestañas superiores para separar "Manuales" de "FAQ".

### Cambiado

- Actualizada la Guía de Bienvenida con instrucciones sobre el buscador.
- Refactorización del archivo `mkdocs.yml` para soportar `mike`.

### Corregido

- Enlace roto en el pie de página que apuntaba al repositorio antiguo.
- Error visual en el modo oscuro del buscador.

## [1.0.0] - 2026-01-10

### Añadido

- Estructura base de la Biblioteca Técnica.
- Manual inicial de soporte para el PDI.
- Configuración de colores corporativos de la ULL.
