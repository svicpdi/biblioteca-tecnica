# Bienvenida

Bienvenido a la sección técnica del repositorio [**Baremación de Plazas: PDI Laboral**](https://sites.google.com/ull.edu.es/soporte-vicpdi/repositorios/baremacion-pdil). En este espacio encontrarás toda la documentación, manuales de usuario y respuestas a preguntas frecuentes relativas a esta temática.

---

## Cómo navegar por esta sección

Para facilitar la navegación, la información se ha organizado en tres grandes bloques que puedes ver en el menú lateral izquierdo:

* **Comunes:** Aquí encontrarás documentación de interés sin importar tu rol. Es un canal que aportará información aclaratoria sobre diversos aspectos.
* **Candidaturas:** Sección dedicada a las candidaturas. Incluye **manuales** de las herramientas, **guías** y **FAQ** para resolver dudas.
* **Comisiones:** Sección dedicada a las Comisiones. Incluye **manuales** de las herramientas **guías** y **FAQ** para resolver dudas.

!!! info "**Consejo**"
    Si buscas un término específico (como "errores"), utiliza la **barra de búsqueda** en la parte superior derecha para localizar rápidamente el documento pertinente.

---

## ¿Necesitas ayuda adicional?

Si tras revisar los manuales y las secciones de FAQ sigues teniendo dudas técnicas, recuerda que puedes [contactar con el soporte](https://sites.google.com/ull.edu.es/soporte-vicpdi/contacto).
